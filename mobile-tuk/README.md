# GoWash Tuk Driver App 🛺

This is the professional mobile application for GoWash Tuk Tuk drivers. It is built using **Expo** and **React Native** with a premium dark-themed design.

## Features
- **Real-time Dashboard**: Overview of daily earnings and ride counts.
- **Availability Toggle**: Switch between 'Online' and 'Offline' modes.
- **Nearby Jobs**: View and accept laundry pickup/delivery requests.
- **Modern UI**: Sleek professional look with amber accents and glassmorphism.
- **Integrated Stats**: Track your rating, distance, and hours online.

## Getting Started

1.  **Install dependencies**:
    ```bash
    cd mobile-tuk
    npm install
    ```

2.  **Start the app**:
    ```bash
    npx expo start
    ```

## Professional Recommendations
- **Navigation**: Integrated with Google Maps for one-tap route guidance.
- **Push Notifications**: Real-time alerts for new job requests nearby.
- **Voice Support**: Hands-free order announcement using Text-to-Speech.
- **Multilingual**: Supports Sinhalese, Tamil, and English.
